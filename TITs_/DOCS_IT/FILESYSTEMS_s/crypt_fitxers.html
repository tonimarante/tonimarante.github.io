Encriptar fitxers

Última modificació: març de 2019

Per a encriptar fitxers, tenim diverses opcions. Algunes:

Encriptar fitxers amb gpg
gpg --output doc.gpg --symmetric fitxer_a_encriptar

Desencriptar amb gpg:
gpg --output fitxer_a_encriptar --decrypt fitxer_encriptat

Encriptar fitxers de text amb vim

Un altre opció simple, per a fitxers de text, es vim. El que farem es:
:X

En demanarà la clau de xifrat. Un cop salvat, vim ens demanarà aquesta clau cada cop que volguem obrir el fitxer.
