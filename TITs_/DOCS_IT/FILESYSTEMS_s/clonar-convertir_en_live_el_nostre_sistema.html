Systemback
Systemback permet crear punts de restauració per fer una instantània del teu sistema en un moment concret del temps. Més endavant, pots utilitzar aquests mateixos punts de restauració per restaurar el teu "sistema" de nou tal com era. Systemback també inclou la creació, a partir d'aquests snapshots, d'un sistema "Live" (pot generar la .iso o "cremar" un pendrive USB directament).
El web oficial és: https://launchpad.net/systemback. A Ubuntu podem afegir el ppa que ens indica i instal·larem la versió 1.8
Alternativament a https://github.com/fconidi/systemback-install_pack-1.9.4 tenim versions 1.9, compatibles amb Debian 10 i Ubuntu 20.04.
