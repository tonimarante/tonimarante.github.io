*Canviar l'etiqueta a un pendrive USB*
Última actualització: desembre 2012

Per sapiguer que està passant quan punxem un pendrive a l'ordinador fem:

dmesg | tail

Aquí veurem si el detecta o no, possibles errades i sota quin /dev l'agafa el sistema. Un cop tenir disc i partició identificat, podrem canviar l'etiqueta.

ATENCIÓ: per a poder canviar l'etiqueta la partició ha d'estar desmuntada.

Dos escenaris:

  • Pendrive amb format FAT:

Instal·lem les mtools:

apt-get install mtools

Suposem que la partició que a la que volem canviar l'etiqueta és /dev/sdc1:

mlabel -i /dev/sdc1 ::boot_puppy

  • Pendrive amb format ext2, ext3 o ext4:

instal·lem e2fsprogs:

apt-get install e2fsprogs

Suposem que la partició que a la que volem canviar l'etiqueta és /dev/sdc1:

e2label /dev/sdc1 boot_puppy
