disc dur, num sèrie

*Obtenir número de sèrie de disc dur*
Última actualització: octubre de 2016

Quan tenim un equip amb varis discs durs i un d'ells s'espatlla, necessitem número de sèrie del disc avariat per a poder indentificar-lo entre els existents.

apt-get install hdparm

Un cop tinguem instal·lat hdparm, hem de fer:

hdparm -i /dev/sda

I obtindrem una sortida com aquesta:
