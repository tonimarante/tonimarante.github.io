Test (destructiu) de discs durs i SD

  • Podem comprovar l'estat d'un disc dur o SD amb aquest test destructiu. Tenir en compte que s'esborraran totes les dades dels discs que testegem amb aquesta instrucció:

badblocks -wsv /dev/<device>
Si volem saber el temps que triga aquest test, podem fer:

time badblocks -wsv /dev/<device>

Comprovar rendiment (velocitat) d'un disc dur

sudo hdparm -t --direct /dev/sda
