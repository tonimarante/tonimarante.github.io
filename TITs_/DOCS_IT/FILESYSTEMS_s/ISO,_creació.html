ISO, creació

:tagiso:
*Crear ISOs*
Última actualització: setembre 2017

Per a crear una ISO a partir d'un CD o DVD, hem de fer:

dd if=/dev/cdrom of=/home/jomateix/elmeucd.iso

Si el CD o DVD està defectuós o protegit podem provar amb dd_rescue. L'insta·larem amb apt-get install ddrescue

Per a fer una ISO a partir d'una carpeta, ens fa falta instal·lar genisoimage. Un cop instal.lat:

genisoimage -o lamevaiso.iso /home/jomateix/carpetaquevullpassaraiso/

Si la carpeta te algun fitxer més gran de 4 GB hem d'afegir el paràmetre -allow-limited-size:

genisoimage -allow-limited-size -o lamevaiso.iso /home/jomateix/carpetaquevullpassaraiso/

Si volem crear una ISO a partir d'un DVD comercial, primer ens hem d'assegurar que el podem reproduir sense problemes. Potser hem d'instal·lar algun paquet necessari per a reproduir DVDs comercials. Un cop el podem reproduir sense problemes, crear la ISO és tan fàcil com:

dd_rescue -A /dev/cdrom Married.iso
