ext2 a ext3

*Convertir sistema de fitxers de ext2 a ext3*
Última actualització: juny de 2017

Per convertir de ext3 a ext2:

 umount /dev/hda3
 tune2fs -O ^has_journal /dev/hda3
 e2fsck /dev/hda3

Un cop fet això, s'ha de reconfigurar /etc/fstab per indicar que el sistema de fitxers ara és ext2.

Un altre manera podria ser:

debugfs -w /dev/hda3
debugfs: features -needs_recovery -has_journal
Filesystem features: dir_index filetype sparse_super
debugfs: quit

Per tornar a convertir ext2 a ext3:

tune2fs -j /dev/hda3

En discs durs que no siguin de sistema podem deshabilitar l'espai reservat pel root amb:

tune2fs -m 0 /dev/hda3
