xrdp

*Instal·lació d'xrdp (accedir a un escriptori Linux des de Windows)*
Última actualització: maig de 2019

Una opció per accedir remotament a un escriptori Linux, és instal·lar xrdp

apt-get install xrdp xorgxrdp

xrdp obre l'últim entorn d'escriptori que hagi utilitzat l'usuari executant /etc/xrdp/startwm.sh.

Si volem connectar amb xrdp i treballar sempre amb un escriptori en particular, per exemple xfce4, hem de modificar el fitxer .xsession propi:

echo xfce4-session > ~/.xsession

o si volem treballar amb i3:

echo i3 > ~/.xsession

Contingut una mica més depurat de .xsession:

#!/bin/sh 
if [ -r /etc/default/locale ]; then 
    . /etc/default/locale 
    export LANG LANGUAGE 
fi 
# default (= ubuntu) 
#. /etc/X11/Xsession 

# unity 2d 
#echo "gnome-session --session=ubuntu-2d" > ~/.xsession 
#. /etc/X11/Xsession 

# xfce 
# startxfce4 
  
i3
exit(0)

Un cop configurat podrem accedir des de qualsevol client com l'escriptori remot de windows o des de remmina o rdesktop des de Linux.
