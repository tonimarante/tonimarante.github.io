w3m

:tagw3m:
Última modificació: agost de 2017

w3m és un navegador web que treballa en mode text. Com sempre passa amb les eines de mode text, sota l'aparença d'una cosa simple s'amaga un navegador potent, que renderitza molt bé les planes que tenen taules, p.e. i que pot mostrar imatges sempre que l'obrim des de l'emulador de terminal xterm.

Llista d'algunes operaciones que es poden fer amb w3m:

*Navegació*
Shift + u Obrir URL
Shift + b Tornar enrere
Shift + r Refresca pàgina

*Tabs*
Shift + t Obrir en un nou tab
Ctrl + t Obre el link en un nou tab
Esc + t Ens mostra tots els tab oberts; seleccionem el que volguem
Shift + [ Tab previ
Shift + ] Tab següent
Ctrl + q Tanca tab

*Hipervincles*
c Mostra URL actual
u Sobre un link, mostra URL a la que apunta
Esc + l Mostra tots els hipervincles de la pàgina
Shift + l Llista amb tots els hipervincles, anchors i imatges

*Imatges*
i Sobre una imatge, mostra la URL d'ella
Shift + i Veure imatge en visor extern
Esc + i Guardar imatge

*Historial*
s Historial sessió actual
Ctrl + h Historial

*Altres operacions:*
Esc + s Salvar html actual a fitxer
Shift + s Salvar renderitzat a fitxer
Esc + e Editar renderizat
v Alterna vista entre html i renderitzada
Shift + m Open page in alternative browser
m Deshabilita ratolí

*Favorits*
Esc + a Afegir favorit
Esc + b Veure favorits
