Debian lleugera

Última actualització: gener de 2021

En ocasions necessitem una Debian "lleugera", menys pesant que la que obtens quan instal·les des del DVD el sistema estàndar.

Instal·larem des del CD de netinst un sistema bàsic. Si pensem treballar en testing o unstable millor baixar la iso de les daily-builds de 32 o 64 bits.

  • http://cdimage.debian.org/cdimage/daily-builds/daily/current/i386/iso-cd/
  • http://cdimage.debian.org/cdimage/daily-builds/daily/current/amd64/iso-cd/

Ara:

    apt-get update
    apt-get upgrade
    reboot
    apt-get install x-window-system-core
    apt-get install fluxbox jwm i3

Si tenim problemes i necessitem reconfigurar les x-org, podem fer:

dpkg-reconfigure xserver-org

De cara a engegar les X tenim vàries opcions:

-Si només fem servir fluxbox, jwm o i3 no és obligatori un gestor de sessions; un cop logats podem executar startfluxbox, jwm o i3. Si només tenim un window manager, podem executar simplement startx
-A pesar de tenir només un window manager, volem un gestor de sessions que ens ofereixi un login gràfic i s'ocupi d'arrecar les X. En aquest cas farem servir xdm:

apt-get install xdm

-Si tenim més d'un window manager i volem poder conmutar entre ells fàcilment podem instal·lar wdm, slim, lightdm o gdm.

Com a window manager lleuger tenim aquestes possibles opcions:

i3 (el meu preferit) spectrwm iceWM fluxbox openbox jwm

Configurant coses de la interfície gràfica

  • Com modificar el DPI de les fonts: Per a no deixar-nos la vista amb les fonts tan petites, podem modificar a gdm la línia que llença el servidor X. La línia original del comandament diu: /usr/bin/X -audit. Canviant-la a /usr/bin/X -dpi 100 -audit obtindrem el resultat desitjat.
  • xbindkeys:

Si per algun motiu tenim problemes assignant tecles a accions des del windows manager, com a últim recurs podem fer servir xbindkeys (http://www.nongnu.org/xbindkeys/xbindkeys.html). /> El seu ús és molt senzill, simplement executar-lo i quedarà en segon pla esperant a llençar les accions configurades i assignades a les tecles que vulguem. Això tant per consola com per les X.
Per a obtenir el codi que torna una tecla quan la piquem amb:

xbindkeys -n -mk
