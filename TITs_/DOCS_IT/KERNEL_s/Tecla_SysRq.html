Tecla SysRq

Última modificació: novembre de 2017

El kernel de Linux "escolta" a la combinació de tecles Alt + SysRq (en alguns teclats aquesta tecla està traduïda com Petsis) i executa en conseqüència instruccions de baix nivell. Pot ser útil, per exemple per apagar o rebootar un sistema del que hem perdut el control, sense corrompre el sistema de fitxers.
Algunes combinacions:

Alt + SysRq + 9 Posar el nivell de logs del kernel a 9 (aquí podem posar un valor del 0 al 9)
Alt + SysRq + s Sincronitzar tots els sistemes de fitxers muntats
Alt + SysRq + b Fer un reboot inmediat sense desmontar cap sistema de fitxers
Alt + SysRq + o Fer un shutdown del sistema
