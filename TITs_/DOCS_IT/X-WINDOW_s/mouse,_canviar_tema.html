*Canviar el tema del punter del ratolí a Debian*
Última actualització: juliol de 2013

Si volem un punter més gran:

apt-get install big-cursor

Si el que volem es canviar el tema del punter:

Instal·lem temes:

apt-get install dmz-cursor-theme

Ara escollim un dels temes (això s'aplica a nivell global, per a tots els usuaris del sistema):

update-alternatives --config x-cursor-theme

Si el tema del punter es escalable (dmz-cursor-theme ho és), podem definir la mida que volem a nivell d'usuari, definint-la en el fitxer ~/.Xresources:

cat ~/.Xresources

Xcursor.size: 32 ! 16, 32, 48 or 64 may also be good values
