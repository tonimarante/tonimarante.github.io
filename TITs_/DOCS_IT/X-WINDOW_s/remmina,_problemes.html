*Problemes de remmina per obrir per rdp servidors windows 2008 R2*
Última actualització: març de 2015

Amb remmina , cíclicament em trobo amb problemes per accedir via rdp a servidors windows 2008 R2. La solució sempre es "tocar" el paràmetre "Seguretat" que es troba a la pestanya "Avançat"

En ocasions el problema es soluciona canviat de "RDP" a "Negocia".
En d'altres ocasions el problema es resol canviat de "Negocia" a "RDP"
