Slim i Lightdm a Debian

Última actualització: setembre de 2019

Slim

Per a instal·lar-ho:
apt install slim

Per a canviar el tema aplicat a slim ho farem a /etc/slim.conf:

current_theme TM_bandes,TM_candau,TM_edifici,TM_laberint

Podem definir diversos temes (separats amb punt i coma) i slim agafarà un aleatoriament.
Els temes estan a la carpeta /usr/share/slim/themes.

En la finestra de login, podem escollir sessió picant a F1. Les sessions disponibles, les llegueix slim dels fitxers existents al directory /usr/share/xsessions/

Lightdm

Per a instal·lar-lo:
apt install lightdm

Si volem posar un tema diferent el que podem fer és instal·lar:
apt install lightdm-settings slick-greeter
Ara podem podrem aplicar fàcilment el tema que volguem o un fons de pantalla personalitzat, executant lightdm-settings.

Per a definir la sessió per defecte, ho farem a
/usr/share/lightdm/lightdm.conf.d/01_debian.conf
aquí afegirem una línia:

user-session=i3

Si volem que i3 sigui la sessió per defecte. Podem obtenir la llista de sessions mirant els fitxers del directori /usr/share/xessions

Al mateix fitxer /usr/share/lightdm/lightdm.conf.d/01_debian.conf, poden configurar el greeter per defecte (el greeter és el que ens proporciona el necessari per a poder logar-nos al sistema des d'un display manager). Hem de modificar la línia:

greeter-session=lightdm-greeter

Podem obtenir la llista de greeters disponibles, mirant els fitxers que hi ha al directori /usr/share/xgreeters/
