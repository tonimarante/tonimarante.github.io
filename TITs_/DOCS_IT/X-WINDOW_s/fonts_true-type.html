fonts true-type

*Instal·lar fonts TrueType*
Última actualització: gener 2020

A Debian instal·larem les fonts escollint el paquet que volguem.
Si volem saber les fonts que tenim disponibles en Debian ho podem mirar a http://fonts.debian.net /> Hem de tenir en compte que si volem instal·lar fonts que no siguin lliures, ens haurem d'afegir el repositori contrib.

Si tenim que instal·lar fons TrueType de manera manual, ho podem fer copiant les fonts a dos llocs diferents:

  • A /usr/share/fonts (necessitarem permisos de root) si volem que estiguin disponibles per a tots als usuaris
  • A ~/.fonts (si no existeix la carpeta, la crearem) per a que estiguin disponibles només per a nosaltres. En aquest cas ho podem fer sense privilegis especials.

Aquestes rutes les podriem modificar si volguéssim a /etc/fonts/fonts.conf, que es el fitxer central de configuració de fontconfig. Podem configurar com renderitzem les fonts del sistema si fem:

dpkg-reconfigure fontconfig-config

Si volem obtenir un fitxer de totes les fonts instal·lades:

fc-list

Si hem de reconstruir la cache del llistat de fonts (això ho farem si un cop instal·lades les nostres fonts no ens aparèixen com disponibles):

fc-cache -fv

Llista de les fonts disponibles per als servidor d'X:

xlsfonts -fn '-----------m'

xlsfonts -fn '-----------c'

SI volem utilitzar les fonts lliures més semblants a les fonts emprades en Windows, son:

  • Asap
  • Carlito
  • Dejavu
  • Liberation
