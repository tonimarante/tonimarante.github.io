raspberry, format SD bootable

*Formatejar una SD des de Linux per a instal·lar una Raspberry Pi*
Última actualització: gener de 2017

Per a instal·lar alguna distribució a la Rasberry a partir de NOOBS o Berryboot primer necessitem formatar la SD d'una manera determinada. Explico aquí com fer-ho amb parted.
Tenir en compte que això esborrarà tot el contingut de la targeta SD.

Si la nostra targeta es /dev/sdb, executem amb sudo o com a root:

parted

mklabel msdos

mkpart primary ext2 1 100%

set 1 boot on
set 1 lba on
q (per sortir de parted)

mkfs.vfat /dev/sdf1

I ja està. Ara despunxem la SD i la tornem a punxar i li copiem a dintre els fitxers de NOOBS o Berryboot.
Amb la raspberry apagada, li posem la SD i tornem a alimentar-la.
