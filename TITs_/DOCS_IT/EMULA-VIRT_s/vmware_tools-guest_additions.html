*Instal.lar les Vmware Tools en Debian i Fedora*
Última actualització: setembre 2022

Open VM tools in Debian

Una bona opció és instal·lar les Open VM tools:
apt install open-vm-tools

Vmware Tools en Debian

Abans, haurem de tenir instal·lat les eines de compilar i les capçaleres del kernel en ús.
1- Instal·lem:

apt-get install make gcc
apt-get install linux-headers-`uname -r`

Alternativament, en comptes de les dues instruccions anteriors podriem fer això; el que més ens agradi:
apt-get install build-essential module-assistant
m-a prepare

2- Muntem l'ISO que conté les tools, descomprimim el tar.gz i executem
vmware-install.pl

3- Per a qualsevol canvi posterior, tornarem a executar
vmware-config-tools.pl

4- Si volem desinstal.lar les tools, executarem
/usr/bin/vmware-uninstall-tools.pl

Vmware Tools en Fedora:
Instal·lem:
yum install perl gcc make kernel-devel

I els mateixos pasos 2, 3 i 4 anteriors

Instal.lar les Guest Additions (Virtualbox) en Debian i Ubuntu

Última actualització: abril 2015

Per a preparar el sistema hem d'instal.lar paquets que ens permetin compilar:

apt-get install build-essential module-assistant

Ara preparem el sistema per a poder construir mòduls de kernel:

m-a prepare

De fet aquests dos pasos anteriors son vàlids/necessaris també per a instal·lar les VMware Tools i les Parallels Tools.

Per a instal·lar les Guest Additions, un cop muntada la ISO que les conté, fem:

VBoxLinuxAdditions.run

Per a desinstal·lar les Guest Additions:

VBoxLinuxAdditions.run uninstall

Nota: per a instal.lar les Guest Additions en Arch, necessitem les linux-headers
pacman -S linux-headers

Instal·lar les Parallels Tools a Scientific Linux 6

Última actualització: agost 2012

Primer instal·lem tot el necessari per a poder compilar els mòduls corresponents a les tools:
yum install gcc kernel-devel make

I instal·lem les tools com sempre. Jo sempre faig:

muto el CD que les conté i copio la carpeta "Parallels Tools" cap a alguna banda al disc dur.
Executo com a root, "install"
Per a desinstal·lar, executem el mateix "install", però escullim fer un "remove" de les Tools.
