*Deshabilitar samba v1 al servidor samba*
Última modificació: juny de 2017

Per a deshabilitar smb v1 del nostre servidor samba, hem de tenir habilitades aquestes dues línies a la secció GLOBAL del nostre fitxer /etc/samba/smb.conf:

min protocol = LANMAN2
max protocol = SMB2
