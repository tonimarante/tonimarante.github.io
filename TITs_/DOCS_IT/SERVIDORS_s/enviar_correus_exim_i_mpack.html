enviar correus exim i mpack

*Enviament de correus de forma automàtica amb Exim, bash, mail i mpack*
Última actualització: maig de 2010

Podem fer que els sistema ens envii correus de manera automàtica. Primer ens farà falta tenir ben configurat exim (https://sites.google.com/site/tonimarante/apunts-de-linux-debian/servidors/servidor-smtp-anonim-amb-exim4) i llavors podem fer:

cat /root/logs/estat_raids.txt | mail -s "Estat raids" la_meva_adreça@gmail.com

Si volem adjuntar fitxers, el més ràpid es instal.lar mpack; un cop el tenim instal.lat podem fer:

mpack -s "Enviament fitxer adjunt" el_fitxer_que_vull_adjuntar.zip la_meva_adreça@gmail.com
