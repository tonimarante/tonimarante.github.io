*Connectar a un equip Linux via serial port*
Última actualització: abril de 2013

Per connectar a una màquina Linux des de port sèrie, ens fa falta un cable d'interconnexió molt senzillet, que només te 3 cables:

db25 db25 db9 db25
2 3 2 2
3 2 3 3
7 7 5 7

Hem d'afegir a /etc/inittab una línia semblant a aquesta:

T0:23:respawn:/sbin/getty -L ttyS0 9600 vt100

Per activar la línia afegida podem fer

'init q'.

o també:

kill -HUP 1

Es poden fer servir el 'getty', el 'agetty'
o qualsevol altre. ttyS0 és el COM1, ttyS1 és COM2, etc.

Per connectar des de win a aquesta màquina Linux podem fer servir qualsevol programa emulador de terminal: telix o el mateix hyperteminal. Podem transferir fitxers amb el protocol zmodem:

-Enviar: fem rb al Linux i enviem el fitxer amb Ymodem
-Rebre: fem sz nom fitxer (o * per enviar tots) i rebem a la màquina connectada amb Ymodem
