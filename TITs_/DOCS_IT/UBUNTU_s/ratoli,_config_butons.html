ratoli, config butons

*Configurar els botons del ratolí*

Última modificació: agost de 2012

La meva pobre mà ha dit basta, i finalment he tingut que canviar els meus ratolins per un 3M-EM500GPL. El canvi ha provocat que vulgui configurar-me la funció dels botons; ho he pogut fer amb xmodmap -e.
Per exemple:

el primer botó com sempre,
el segon (botó dret) vull que es comporti com si fos el tercer (botó del mig)
el tercer vull que es comporti com el segon:

xmodmap -e "pointer = 1 3 2"

Un altre exemple:

el primer botó com sempre,
el segon (botó dret) vull que es comporti com si fos el tercer (botó del mig)
el tercer vull que no faci res:

xmodmap -e "pointer = 1 3 0"

M'he creat un script-diàleg gràfic. Només funciona si tenim zenity instal·lat:

if zenity --question --text "Estàs treballant amb el 3M-EM500GPL?" ;
	then
		xmodmap -e "pointer = 1 3 0"
		#xmodmap -e "pointer = 1 3 2"
	else
		xmodmap -e "pointer = 1 2 3"
fi
