LP a mp3

*Passar LPs o cassettes a mp3*
Última actualització: abril de 2012

1- Gravarem el que es entri a la targeta de so, durant 1 hora amb:
sound-recorder -c 2 -S 60:00 -f wav -k 1.wav

2- Retallem automaticament les pistes amb el programari gramofile

3- Encodem tots els wav a mp3 amb lame:

	#! /bin/bash
	for i in *.wav;
	do lame "$i" "$i".mp3
	done

Si ens agrada més bladeenc...

	#! /bin/bash
	for i in *.wav;
	do bladeenc $i $i.mp3
	done

4- Si ens fa falta tornar a wav pel que sigui:

#! /bin/bash
for i in *.mp3;
do lame --decode $i $i.wav
done
