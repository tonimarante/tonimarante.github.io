Afegir música a iPod Classic des de Linux

Última actualització: març de 2019

Per a que Linux pugui muntar un iPod en mode d'escriptura, el sistema de fitxers de l'iPod (hfs+) no ha de tenir journaling. Des de un Mac hem de fer:

diskutil list

Per a saber quin es el disc i partició del iPod. I ara:

diskutil disableJournal /dev/xxx

Ara des de Linux i el Rythmbox, p.e., podrem passar-li música.

Possible problema: no es munta. El problema és que el sistema de fitxer s'ha desmuntat malament.
Podem reparar-ho des del mateix Linux. Si la partició és /dev/sdb3, p.e., farem:

sudo fsck.hfsplus -r /dev/sdb3
