Grub està un disc o no

:taggrub:
*Comprovar si grub està instal·lat a un disc*
Última modificació: maig de 2018

Podem mirar si grub està grub a un disc examinant els primers the 512 bytes del disc.
Si suposem que el nostre disc és /dev/sda:

dd bs=512 count=1 if=/dev/sda 2>/dev/null | strings

El resultat serà tal com:

ZRr=
`|f
\|f1
GRUB
Geom
Hard Disk
Read
Error

Veiem llavors que GRUB està instal·lat.
